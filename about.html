About TaskFlow

TaskFlow is a productivity dashboard designed to help users manage tasks, track deadlines, and improve workflow efficiency.

Key Features

  • Task creation with priority and scheduling
  • Start and due date tracking
  • Live countdown timers
  • Overdue detection with visual alerts
  • Task editing and completion tracking
  • Productivity metrics and completion rate

Why I Built This

This project was built to demonstrate real-world frontend development skills, including state management, dynamic UI updates, and user-focused design. It reflects how modern productivity tools are structured and optimized for usability.

Tech Stack

HTML, CSS, JavaScript, LocalStorage
